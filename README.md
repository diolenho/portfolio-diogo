## 🧠 Sobre

Este projeto foi criado como uma landing page de portfólio para reunir experiências profissionais, projetos e iniciativas relacionadas a Product Management, UX Research e produtos digitais.

O objetivo é apresentar de forma simples e acessível:

- trajetória profissional  
- projetos desenvolvidos  
- visão estratégica de produto  
- experiências em discovery e UX  
- habilidades técnicas complementares  

---

## 🎯 Destaques

- Estruturação de portfólio focado em Product Management  
- Integração com projetos e cases desenvolvidos no Notion  
- Formulário funcional utilizando Formspree  
- Publicação via GitHub Pages  
- Desenvolvimento e deploy realizados de forma independente
